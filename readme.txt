Simple application created in vanilla JavaScript (ES5,ES6) and jQuery to utilize Flickr API to show photos.

You can search photos by any text.

List of images show theirs square thumbnails.

After clicking thumbnail you will get larger version of this photo with EXIF information.


Prerequisites

You will need the following things properly installed on your computer.
Node.js (with NPM) - version 6 or higher
npm install
gulp

Running / Development
Visit app at http://localhost:3000.
